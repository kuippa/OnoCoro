# PLATEAUタワーディフェンス
## Unityのあたらしいプロジェクト
## 期間
	３～６ヶ月 作業１５０時間ぐらい / 前回のニホンゴムズイでは３００時間ぐらい

## ゴール
	実際の街並みや地形に即したタワーディフェンスをつくってリリースする

## 目論見 
	ローカルインフラの大切さを実感してもらう
	将来的にまちづくりに利用できるデータ群をつくる

## 展開（今回はやらないけど念頭にはおく）
	災害データのインポート
	防犯データのインポート
	ハザードマップのインポート
	国土地理院細密地形データ
	後につくるオープンクラフト系シミュレーションにも技術的資産として流用できるように
	町会運営、自治体運営に役立つようなデータをためておく
	街角消化器などの設置計画につかえるようにエクスポート
	街路灯位置などリアルロケーションデータを溜め込める仕組み
	PLATEAUなど未対応な地域についてもuser generated content（UGC）
	シビックテック、クラウドソーシング

## 挑戦すること
	GooglePlayStoreで配布する / 前回はAppleStore
	簡単な計画、設計と疎結合なプログラム / 前回は設計なしで何ができるのか試しながら作った
	遊べる程度のゲームバランス / 前回は遊ぶには値しなかった

## 技術的挑戦
	Unity3D、カメラ操作、WASP移動・マウスなどの操作
	Android版
	Plateau
	OpenStreetMap
	GISデータ
	CityGML（まだ早いか？）
	携帯などのGPS対応
	Firebase/Cloud Firestore/Cloud Functions for Firebaseなどの外部API利用
	すこしだけ設計 / 前回場当たり的な密結合だったのですこしだけ疎結合に
	外書きファイルにセーブ / Unityのデフォセーブ機能から

## やらないこと
	Blenderとかな3Dデザイン学習
	複数人での開発、ソースのバージョン管理(Plastic SCM) / 開発コストをあげないため
	かっちりした設計、テスト自動化
	モノビヘイビアーを使わないほどの分離、モジュール化
	asmdef(Unity Assembly Definition)
	広告やプレイヤーを増やすための施策（保守、運用に割くリソース）
	点群GIS（やりたい）
