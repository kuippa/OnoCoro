# OnoCoro - プロジェクト概要

**OnoCoro** (オノコロ) は、国土交通省の PLATEAU (プラトー) プロジェクトの3D都市モデルを活用した、まちづくりタワーディフェンスゲームです。

---

## プロジェクトの目的

### 主な目標

1. **地理情報の可視化**: PLATEAU SDK を使用して、実際の都市データをゲーム内で表現
2. **教育的価値**: まちづくりやインフラの重要性を楽しみながら学べる
3. **技術実証**: Unity + PLATEAU SDK の組み合わせによる地理情報ゲームの可能性を探る
4. **オープンソース**: SSD故障の教訓から、オープンソースでバックアップと知見共有を実現

### ターゲットユーザー

- タワーディフェンスゲームファン
- 地理情報・都市計画に興味がある人
- PLATEAU SDK を使った開発事例を探している開発者

---

## ゲームコンセプト

### 基本ゲームプレイ

**ジャンル**: タワーディフェンスゲーム

**コンセプト**: 実在の都市をフィールドとして、インフラを配置しながら街を守る

### 特徴

1. **実際の地理データを使用**
   - PLATEAU プロジェクトの CityGML データを読み込み
   - 実在の建物や地形をゲームフィールドとして利用

2. **まちづくり要素**
   - インフラ（防衛タワー）の配置
   - リソース管理（予算、エネルギーなど）
   - 都市の発展と防衛のバランス

3. **ステージエディタ**
   - カスタムステージの作成
   - YAML形式での設定管理
   - 地理データの読み込み設定

---

## プロジェクトの非目的

このプロジェクトは以下を目的としていません:

### ❌ 商業リリース

- 現時点では商業リリースを目指していません
- 技術実証とオープンソースコミュニティへの貢献が主目的

### ❌ 完全なシミュレーション

- リアルな都市シミュレーションではありません
- あくまでゲームとしての面白さを優先

### ❌ プロフェッショナルなGIS

- GIS（地理情報システム）の専門ツールではありません
- PLATEAU SDK の活用事例として位置づけ

## プロジェクトの背景

### SSD故障からの復旧プロジェクト

**重要な経緯**:

- **2024年**: 開発中だったプロジェクトのSSDが物理的に故障
- **2年間の空白**: 復旧不可能と判断し、プロジェクトは御蔵入り
- **2026年元旦**: 2年前のバックアップを発見し、復旧作業を開始
- **オープンソース化**: 同じ悲劇を繰り返さないため、GitHubで公開管理

### 教訓

1. **バックアップの重要性**: 定期的なバックアップは必須
2. **Git管理**: ローカルだけでなくリモートリポジトリでの管理
3. **オープンソース**: 公開することで複数の場所にバックアップ
4. **ドキュメント**: 復旧時に必要な情報を残す

## 技術スタック

### ゲームエンジン

- **Unity 6.3**: 最新のUnityエンジンを使用

### 主要ライブラリ

- **PLATEAU SDK for Unity**: 3D都市モデルデータの読み込み・表示
- **Cinemachine**: カメラ制御
- **glTFast**: 3Dモデルのロード
- **Unity Input System**: 入力管理

### 開発言語

- **C#**: Unity スクリプティング

## FAQ

### Q: ゲームはプレイできますか？

**A**: 現在は復旧作業中で、完全にはプレイできません。Phase 3 完了後にプレイ可能になる予定です。

### Q: PLATEAU SDK なしでも動作しますか？

**A**: いいえ、PLATEAU SDK はこのプロジェクトの中核機能です。削除や置き換えはできません。

### Q: 商用利用は可能ですか？

**A**: MIT ライセンスのため、商用利用も可能です。ただし、PLATEAU データの利用規約を別途確認してください。

### Q: macOS や Linux でも開発できますか？

**A**: Unity 自体は macOS/Linux に対応していますが、プロジェクトの運用スクリプトは Windows PowerShell を前提としています。

### Q: なぜオープンソースなのですか？

**A**: SSD故障でプロジェクトを失った経験から、同じ悲劇を繰り返さないため、そして知見を共有するためです。
