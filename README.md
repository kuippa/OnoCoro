# OnoCoro
国土交通省のPlateauを使ったタワーディフェンスゲームになる予定。
2024年の2年前バックアップから復旧最中のUnityプロジェクトです。
SSDが吹き飛んだため御蔵入りしていましたが、復活させようと2026年元旦に再始動しました。
バックアップとっていないかったことを反省してオープンソースでgithubに公開することにしました。
しばらくは復旧作業になります。
